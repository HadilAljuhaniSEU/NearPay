# NearPay
Smart Financial Relationship Management for Local Businesses.

NearPay is a FinTech mobile application that enables small and medium-sized businesses to manage customer debts digitally through a secure, AI-powered platform. The project focuses on debt management, payment tracking, smart reminders, and financial insights.

## Features
- Digital debt management
- Merchant dashboard
- Customer debt review
- Full & partial payment support
- Smart payment reminders
- AI-powered financial insights
- Trust Score for customers
- Payment history tracking
- Financial analytics dashboard
